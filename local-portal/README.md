# Local portal (dev home)

Single-page hub styled like [myskillsplus.com](https://myskillsplus.com/) so you can open **SkillMap**, **Vani**, **Employer**, **Individual**, and other local apps from one place.

## Run

```bash
cd local-portal
npm run dev
```

Open **http://localhost:3099/** (all app links use `http://localhost:<port>` so they match a Vite/Next server bound to `localhost`.)

Edit default ports in `index.html` (the `PORTS` object at the top of the script) if your apps use different ports.

Production marketing site: [https://myskillsplus.com/](https://myskillsplus.com/)
